# Team Project - BookBook

### 진행 기간
2023-10-31 ~ 2024-12-08

## 소개
Bookbook은 사용자의 취향에 맞는 책을 추천해주는 도서 추천 사이트입니다. 
사용자는 키워드로 책을 검색하고, 심플한 카드 형식으로 책의 핵심 정보를 확인할 수 있습니다.
